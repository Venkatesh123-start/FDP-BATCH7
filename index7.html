HTML Basics

HTML is the language used to create the structure of web pages. Every website starts with HTML!

What is HTML?

HTML stands for HyperText Markup Language. It uses "tags" (enclosed in angle brackets) to tell browsers what to display.

Basic HTML Structure

Every HTML document follows this structure:

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> <h1>Hello World!</h1> </body> </html>

üîç Let's break it down:

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element containing everything
  • <head> - Contains metadata (title, links, scripts)
  • <body> - Contains the visible content

HTML Tags

Tags come in pairs: an opening tag and a closing tag. The content goes between them:

<tagname>content</tagname>

Examples:

  • <h1>This is a heading</h1>
  • <p>This is a paragraph</p>
  • <a href="url">This is a link</a>
üí° Important: Most tags need to be closed with a closing tag (like </p>). Forgetting to close tags is one of the most common mistakes!

Common HTML Elements

Let's learn about the most commonly used HTML elements for creating web pages.

Headings

Headings help organize content. There are 6 levels (h1 to h6):

<h1>Main Heading</h1> <h2>Subheading</h2> <h3>Sub-subheading</h3>

Paragraphs

Use the <p> tag for text content:

<p>This is a paragraph of text.</p>

Links

Create links with the <a> tag:

<a href="https://www.example.com">Click here</a>

Images

Add images with the <img> tag (note: no closing tag):

<img src="image.jpg" alt="Description">

Lists

Create organized lists:

<ul> <li>Item 1</li> <li>Item 2</li> </ul> <ol> <li>First</li> <li>Second</li> </ol>

Buttons

Create clickable buttons:

<button>Click Me!</button>
üí° Tip: The alt attribute in images is important for accessibility. It describes what the image shows if it can't be displayed.

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements and make web pages look beautiful!

What is CSS?

CSS controls the appearance of HTML elements: colors, sizes, spacing, fonts, and layout.

Three Ways to Use CSS

1. Inline CSS (in HTML tags):

<p style="color: blue; font-size: 18px;">Blue text</p>

2. Internal CSS (in <head> tag):

<style> p { color: blue; font-size: 18px; } </style>

3. External CSS (separate .css file):

<link rel="stylesheet" href="style.css">

CSS Syntax

CSS follows this pattern:

selector { property: value; property: value; }

Example:

h1 { color: red; font-size: 32px; text-align: center; }
üí° Best Practice: Use external CSS files for better organization and to keep HTML and styling separate.

Common CSS Properties

Learn the most important CSS properties used to style web pages.

Text Properties

color: red; /* Text color */ font-size: 18px; /* Text size */ font-family: Arial; /* Font type */ font-weight: bold; /* Text weight */ text-align: center; /* Alignment */

Background

background-color: blue; /* Background color */ background-image: url(); /* Background image */

Box Model (Spacing)

margin: 20px; /* Space outside element */ padding: 15px; /* Space inside element */ border: 2px solid; /* Border around element */ width: 300px; /* Element width */ height: 100px; /* Element height */

Visual Effects

border-radius: 10px; /* Rounded corners */ box-shadow: 0 4px 6px; /* Shadow effect */ opacity: 0.5; /* Transparency */

JavaScript Basics

JavaScript is a programming language that adds interactivity and logic to web pages!

What is JavaScript?

JavaScript allows you to make web pages interactive. You can respond to user actions, validate forms, animate elements, and much more!

Adding JavaScript to HTML

Using <script> tag in HTML:

<script> console.log("Hello JavaScript!"); </script>

Variables

Variables store data values. Use let, const, or var:

let name = "John"; let age = 25; const PI = 3.14; var city = "New York";

Data Types

let text = "Hello"; // String let number = 42; // Number let decimal = 3.14; // Decimal let isTrue = true; // Boolean let nothing = null; // Null let undefined_var; // Undefined

Basic Operations

let x = 10; let y = 5; let sum = x + y; // Addition: 15 let difference = x - y; // Subtraction: 5 let product = x * y; // Multiplication: 50 let quotient = x / y; // Division: 2

If Statements

Make decisions based on conditions:

let age = 18; if (age >= 18) { console.log("You are an adult!"); } else { console.log("You are a minor."); }

Functions

Reusable blocks of code:

function greet(name) { return "Hello, " + name + "!"; } greet("John"); // Returns "Hello, John!"

JavaScript & the DOM

The DOM (Document Object Model) allows JavaScript to interact with HTML elements on the page.

What is the DOM?

The DOM is a representation of your HTML document as a tree of objects. JavaScript can modify this tree to change the page dynamically!

Selecting Elements

Find HTML elements using JavaScript:

// By ID document.getElementById("myId"); // By class name document.getElementsByClassName("myClass"); // By tag name document.getElementsByTagName("p"); // Using CSS selectors document.querySelector(".myClass"); document.querySelectorAll("p");

Changing Content

// Change text document.getElementById("myId").textContent = "New text"; // Change HTML document.getElementById("myId").innerHTML = "<strong>Bold text</strong>"; // Change style document.getElementById("myId").style.color = "red";

Event Listeners

Respond to user actions like clicks:

let button = document.getElementById("myButton"); button.addEventListener("click", function() { alert("Button was clicked!"); });

Common Events

  • click - When element is clicked
  • mouseover - When mouse hovers over element
  • mouseout - When mouse leaves element
  • input - When text input changes
  • submit - When form is submitted
