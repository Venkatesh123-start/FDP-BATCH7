# B Venkatesh Portfolio

A clean, responsive single-page website with a dark hero, project highlights, and social CTAs (GitHub + LinkedIn).

## Preview
Open `index.html` in your browser. On Windows:

```bash
start index.html
```

## Customize
- Update the headline/subtitle and social links in `index.html`.
- Swap project cards under `#projects` with your own work and links.
- Adjust colors and spacing in `styles.css`.
- Tweak CTAs (GitHub/LinkedIn) in the hero and contact sections as needed.

## Notes
- Uses the Inter font from Google Fonts.
- Smooth scrolling enabled for in-page navigation.
